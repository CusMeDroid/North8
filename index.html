Konfigurasi

Ikuti langkah-langkah mudah ini untuk menyiapkan North8 Anda.

Fungsi AI

Buat api key Gemini Di sini, lalu salin dan tempel API Gemini ke North8 App.

Pesan ke Gambar AI

Buat api key Di sini, lalu salin dan tempel API ke Konfigurasi AI di North8 App.

Google Drive

Hubungi developer melalui WhatsApp atau Email agar email Anda ditambahkan ke dalam lingkungan North8 App.

Gmail

Pastikan akun Google Anda sudah terverifikasi 2 langkah, lalu cari sandi aplikasi atau klik Di sini, buat nama aplikasi "North8", lalu salin dan tempel sandi aplikasi di North8 App.

Telegram Bot

Tutorial cara pembuatan bot Telegram ada Di sini.

Cara Menggunakan

Ikuti langkah-langkah sederhana ini untuk memulai.

  • Klik Logika Tertulis untuk sebuah prompt, Anda juga dapat menambahkan gambar.
  • Pilih Elemen UI, misalnya Send Email. Jika sudah muncul di kanvas, klik untuk konfigurasi dan sejajarkan kode agar rapi.
  • Visual Node: Logika Tertulis > AI > Send Email > Display.
  • Lalu klik tombol Jalankan Alur.
